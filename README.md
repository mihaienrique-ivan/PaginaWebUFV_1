# PaginaWebUFV_1
## Descripción del Trabajo

Sitio web personal desarrollado para la asignatura Fundamentos de la Ingenieria Informatica en la Universidad Francisco de vitoria. El proyecto documenta el aprendizaje del estudiante mediante 7 páginas HTML con navegación consistente, diseño responsivo y contenido académico. Tecnologías utilizadas: HTML5, CSS y GitHub.

## Estructura del Proyecto

El proyecto está organizado con la página principal index.html en la raiz, la carpeta public contiene todas las páginas secundarias (about.html, degree.html, fii.html, topic.html, net.html, contacts.html), la carpeta css almacena los estilos en styles.css, y la carpeta images guarda las imagenes del sitio.

## Problemas Durante el Desarrollo

El principal problema fue determinar las rutas correctas para las imagenes. Las imagenes en la carpeta images necesitaban rutas diferentes según la ubicación del HTML. Pero al final descubri que con /Images/ seguido del nombre de la imagen se ubica bien . También fue necesario ajustar las rutas de los enlaces de navegación usando rutas relativas consistentes.

## Conclusiones

Este proyecto me dio conocimientos en estructura HTML , CSS responsivo, rutas relativas en proyectos web y control de versiones con Git. Una buena organizacion desde el inicio evita problemas posteriores y facilita el mantenimiento del código. El desarrollo demuestra la importancia de los fundamentos en ingeniería informática.
